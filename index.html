About Us

Information is exchanged between systems in today's digital world, and each contact/transaction between systems is supposed to be safe and trustworthy. The security of healthcare data can be protected using blockchain technology. It is critical to have an access control system in place that permits only authorized individuals to share medical information. We provide a blockchain-based solution for securely sharing health records in this paper. A smart contract-based data protection and accessibility mechanism is included in the system.

Blockchain Technology

Blockchain operates by generating decentralized distributed ledgers that are disseminated across all system devices. It allows all nodes in the network, primarily transacting nodes and validating nodes, to share records based on peer-to-peer replication and processing.

Ethereum & Smart Contracts

Ethereum is an open-source public blockchain. It uses a distributed computing network for constructing Decentralized Applications. Ethereum helps to create smart contracts and decentralized applications without any middleman. Smart contract is a logic used to develop blockchain applications. Smart contracts are like real contracts which enforce the agreement between two parties over a specific time for a specific charge. In fact, smart contracts are computer programs that can be used to carry out transactions between any two parties using the contracts programmed internally.

Why to choose dApp ?

Main advantages of dApp is that it center around the program's ability to safeguard user privacy. With decentralized apps, users do not need to submit their personal information to use the function the app provides. DApps use smart contracts to complete the transaction between two anonymous parties without the need to rely on a central authority. A decentralized platform can be resistant no single participant on the blockchain can delete messages or block messages from being posted.
